# Probability Fundamentals

## Binomial Distribution
Formula: P(X=k) = C(n,k) × p^k × (1-p)^(n-k)
- n: number of trials
- k: number of successes
- p: probability of success

Common mistakes:
- Confusing n and k
- Forgetting to calculate combination C(n,k)
- Using wrong probability (p vs 1-p)

## Conditional Probability
P(A|B) = P(A ∩ B) / P(B)

## Bayes' Theorem
P(A|B) = [P(B|A) × P(A)] / P(B)
